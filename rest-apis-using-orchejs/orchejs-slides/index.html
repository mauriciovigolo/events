    Desenvolvimento REST em Nodejs - Orchejs

    Desenvolvimento do mesmo projeto de APIs do sorteio das camisetas com Orchejs.

    O que o Orchejs REST oferece para solucionar os problemas apontados anteriormente?

  • Typescript - checagem estática de código, classes e interfaces.
  • A estrutura de pastas pode ser definida de acordo com a equipe - features | por tipos
  • Carregamento das rotas fica facilitado devido o uso de decorators: @Route
  • As validações são realizadas de forma assíncrona pelos validators.
  • O log da biblioteca pode ser usado pela aplicação.
  • Classes genéricas para Request e Response
